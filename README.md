# 🧶 Cozy Yarn Academy

A warm, welcoming online academy for yarn lovers of every skill level — built by crafters, for crafters.

## 🌐 Live Site

[View the website here](https://alimurunganne.github.io/cozy-yarn-academy)

## 📄 Pages

- **Home** – Welcome page with intro video and explore cards
- **About Us** – Mission, vision, features, and community stats
- **History** – The history of crochet through the centuries
- **Tutorials** – Course guide table and video tutorial
- **Contact Us** – Contact info, social links, and enrollment form

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox & Grid, responsive design)
- Vanilla JavaScript
- Google Fonts – [Outfit](https://fonts.google.com/specimen/Outfit)

## 📱 Responsive

Fully responsive across all screen sizes — mobile, tablet, iPad, and desktop.

## 🚀 Deployment

Deployed via **GitHub Pages** from the `main` branch.
